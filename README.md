# q-tac-toe
A minimal static site that displays "hello world".

## Running locally

Start a simple local web server (Python 3 recommended):

```
python3 -m http.server 8000
```

Open `http://localhost:8000` in your browser — the page will display "hello world".

## Files

- `index.html` — main page (shows "hello world")
- `assets/css/style.css` — basic styling
